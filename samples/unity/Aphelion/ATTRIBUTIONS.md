## Music & body SFX — generated in-repo (no license)

`Assets/Audio/Music/*.wav` (Aphelion theme, explore/tension/combat stems) and `Assets/Audio/SFX/*.wav` (heartbeat, pickup chime, UI blip) are **synthesized deterministically** by [`Tools~/AudioGen`](Tools~/AudioGen/Program.cs) in this repo. Regenerate with:

```powershell
cd Tools~/AudioGen
dotnet run -- ../../Assets/Audio
```

D minor, 90 BPM; the three stems share one grid and progression so they layer per the adaptive-music design.

## Adding assets (the rule)

1. CC0 only, from a source page that states the license — link it here.
2. Or generate it with committed code.
3. One row in this file per asset (or per pack subset), added in the same commit as the asset.
